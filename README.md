# realtime-chat

TLDR 
This is a simple realtime(low latency) chat app - moderately scalable

Features 
1. Do 1:1 chat
2. Create group chat
3. Add/remove members to group chat
4. Signin/Singup
5. Guest Signin

```
Specs 
Frontend : Vite + React, 
Backend : socketIO, express
DB : Mongodb
```
